Booleans

What we call boolean values are two constant objects that can never change.
They are use to represent truth values. They are written True and False.

Strings

In programming a string is a representation of text. Every time we want to display something on the screen we need to write our text as a string. To do that we need to encapsulate our text with single or double quotes exactly like this: "This is a Python string".

Integers

Integers in programming are basically numbers. Numbers don't need to be encapsulated like text so you can simply write the number you want and it will be displayed on the screen.
You can do all the classical operation you learned at school with the integer.
There are 4 main arithmetic operators we're going to focus on: Addition, Subtraction, Division, Multiplication. In addition to this, there is also a modulo operator : % which is to the left over of an division.
So for example 5 % 3 = 2. Finally, you can also use exponents in Ruby by Python **.
For example : 2 ** 3 = 8

Variables

One of the most basic concepts in computer programming is a variable. You can think of a variable as a word or name that grasps a single value.
For example, let's say you need to use the the number 33 multiple times, you can just store it in a variable and that variable will remember the number 33 for you.
To create a variable you simply need to give a name to the variable, put an equal sign on its right and to write the data that you want to store after the equal sign.
Example: name = "Codagram" name is the variable which is storing the "Codagram" string.
Or: age = 8 age is the variable which is storing the integer 8.

print

The print function just take whatever you give it and prints it to the screen. It allows you to output what you want to be display.
In fact displaying "Hello World" on the screen is super simple. You just need to use the print keyword and write some parentheses on its right, inside of which you will writte whatever you would like to displayed.
To ouput Hello World you can just do: print("Hello World") and it will appears on the screen.

The len() function

Functions are used to give some behaviour to the object we are working with. We say that we are calling a method on an object. Function can be called on an object with a . or passing the objedt wanted between parentheses. It depend of the function
If you have a string and you feel lazy to count how long is that string, then comes in handy the len() function. You can simply use this method to output the exact number of characters contained in that string.
Example: we have "Hello World", to count the length of this string we can simply do: len("Hello World"). the length is 11 (it also counts spaces).
Pretty simple, right?

Lists

A list is a collection which is ordered anc chanegable. An example of a variable storing a list would be l = [195, "Hello", False]. Lists can store any type of data. This one stores an integer, a string and a boolean value.
To acces to an item inside it's super easy. If you do a[0], the output would be 195 because list's index starts from 0 and count onwards.

You can interact with the list in a number of different ways. You could add a value into a list by using the .append function. You can also remove values by ising the .remove() function. You just have to write into the parentheses what value you want to remove.
For example if we do l.append("cool"), the new list would be l = [195, "Hello World", False, "cool"].
If we do l.remove("Hello World"), the new list would be l = [195, False, "cool"].

Dictionaries

Dictionaries are similar to lists in that they hold data.
However, they are using a key/value pair..
You can access a specific value in a dictionary by searching up its key.
An example of this would be : d = { "Mickey" => "mouse", "Donald" => "duck", "Jasmine" => "human"}.
If we type : print(d["Mickey"], the output will be mouse.

The '.length' method

Methods are used to give some behaviour to our programs. If you have a string and you feel lazy to count how long is that string, then comes in handy the '.length' method. You can simply use this method to output the exact number of characters contained in that string. Example: we have 'Hello World', to count the length of this string we can simply do: 'Hello World'.length and it will output the number 11 (it also counts spaces). Pretty simple, right?

The .replace() function

The replace() function is called the same way on an object with a lenght ans by passing between the parentheses first what you want to replace, and second what will it be replced by.
For examples, if we have our string "Hello World" and that for whatever reason we want to replace the "H" by a "J", we can use the .replace() function. Like this :
"Hello World".replace("H", "J"). The string will become "Jello World".

.upper() & .lower() functions

Let's see two methods. As you might have guessed, the .upper() and .lower() functions convert a string to ALL UPPER CASE or all lower case, respectively. Example: "Hello World".upper() will output "HELLO WORLD".
